AI on All About Berlin? Yes, but not like that.

I am releasing a tool that updates the website when certain facts change. For example, if the minimum wage goes up, it updates the number everywhere on the website. Constant, automated fact-checking.

The best part is that it barely uses AI. A tiny LLM extracts values from text. After that, it's all boring technology. If a value changes, a constant gets updated in the code, a change request is submitted, and a human has to approve it. There are no unsupervised changes, and all content is still written by a human.
